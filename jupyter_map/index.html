This map was generated by the code in this Jupyter notebook, which was a big team effort by Stuart Geiger, Jamie Whitacre, Matthias Bussonnier, and Nami Saghaei. It is based on data from all GitHub users who have committed to any of the repositories in the jupyter, ipython, jupyterlab, or jupyter-attic organizations. The notebook geocodes the data in the location field of each user's profile. Some people leave these blank (about half of all contributors, who are not shown), and for others these are vague (like just having just a country name puts that user in the geographic center of that country).
